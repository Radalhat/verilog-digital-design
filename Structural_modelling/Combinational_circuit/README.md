# Structural Modeling

This folder contains digital circuits implemented using **Structural Modeling** in Verilog.

In structural modeling, a larger circuit is built by connecting smaller modules together. Instead of describing the circuit with `assign` statements or `always` blocks, the design is created by instantiating existing modules and wiring them together.

The projects in this folder helped me understand how complex digital circuits are built from simpler building blocks.

## Contents

The folder includes projects such as:

- Half Adder
- Full Adder
- Ripple Carry Adder
- Multiplexers
- Comparators
- Other structural designs

Each project folder contains:

- Verilog design file
- Testbench
- Waveform
- README

## Tools Used

- Verilog HDL
- Icarus Verilog
- GTKWave
- Visual Studio Code

## What I Learned

Working on these projects helped me understand module instantiation, port mapping, and hierarchical design. It also gave me more practice writing testbenches and verifying the output using waveforms.
