# Comparator (Gate-Level Modeling)

## Overview

This folder contains the gate-level implementation of comparators using Verilog HDL.

The designs use gate-level modeling to compare two binary numbers and determine whether one number is greater than, less than, or equal to the other. I created these projects to understand how comparators work and improve my Verilog coding skills.

## Comparator

A comparator is a combinational logic circuit that compares two binary numbers and indicates whether one number is greater than, less than, or equal to the other.

The following comparator designs are included:

* 1-Bit Comparator
* 2-Bit Comparator
* 4-Bit Comparator

## Design

The comparators are implemented using Verilog gate primitives.

Each design is tested with different input combinations to verify that the outputs match the expected comparison results.
