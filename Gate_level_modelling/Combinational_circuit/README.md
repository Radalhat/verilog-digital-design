# Combinational Circuits (Gate-Level Modeling)

## Overview

This folder contains combinational circuits implemented using gate-level modeling in Verilog.

The designs are built using Verilog gate primitives such as `and`, `or`, `not`, `nand`, `nor`, `xor`, and `xnor`. Working on these circuits helped me understand how different logic gates can be combined to build larger digital designs.

## Contents

The folder includes:

- Multiplexer
- Demultiplexer
- Encoder
- Decoder
- Comparator
- Half Adder
- Full Adder
- Half Subtractor
- Full Subtractor
- 4-bit Ripple Carry Adder

Each project folder contains:

- Verilog design file
- Testbench
- Waveform
- README
