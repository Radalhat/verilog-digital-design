# Combinational Circuits using Verilog (Dataflow Modelling)

## Overview

This repository contains the implementation of common combinational circuits using Verilog HDL with the Dataflow modelling style.

Each circuit includes:
- Verilog design file
- Testbench
- Simulation output using $monitor

## Contents

### Logic Gates
- AND Gate
- OR Gate
- NOT Gate
- NAND Gate
- NOR Gate
- XOR Gate
- XNOR Gate

### Multiplexers
- 2×1 Multiplexer
- 4×1 Multiplexer

### Demultiplexers
- 1×2 Demultiplexer
- 1×4 Demultiplexer

### Adders
- Half Adder
- Full Adder

### Subtractors
- Half Subtractor
- Full Subtractor

### Comparators
- 1-Bit Comparator
- 2-Bit Comparator

## Simulation Tool

- Synopsys Verdi

## Language

- Verilog HDL

## Modelling Style

- Dataflow Modelling
