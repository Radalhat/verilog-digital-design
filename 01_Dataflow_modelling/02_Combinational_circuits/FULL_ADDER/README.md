# Adders

## Overview

An adder is a combinational circuit used to perform binary addition.

This project implements adders using **Verilog HDL** with the **Dataflow Modelling** technique.

## Implemented Circuits

- Half Adder
- Full Adder

## Project Structure

```
Adders/
│── half_adder.v
│── half_adder_tb.v
│── full_adder.v
│── full_adder_tb.v
└── README.md
```

## Features

- Dataflow Modelling
- Separate testbench for each design
- Simulation using `$monitor`
- Well-commented and easy to understand

## Simulation Tool

- Synopsys Verdi

## Language

- Verilog HDL

## Learning Outcomes

After completing this project, you will be able to:

- Understand binary addition.
- Differentiate between half adders and full adders.
- Design adders using Boolean expressions.
- Verify adder functionality using a Verilog testbench.
