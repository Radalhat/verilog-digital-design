# Half Subtractor

## Overview

A Half Subtractor is a combinational circuit that subtracts one binary bit from another. It produces a **Difference** output and a **Borrow** output.

This project implements a **Half Subtractor** using **Verilog HDL** with the **Dataflow Modelling** technique.

## Inputs

- A
- B

## Outputs

- Diff (Difference)
- Borrow
