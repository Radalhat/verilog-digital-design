# Full Subtractor (Behavioral Modeling)

## Overview

This folder contains the behavioral implementation of a Full Subtractor using Verilog HDL. The circuit subtracts two single-bit binary numbers along with a borrow input and produces the Difference and Borrow outputs.

## What is a Full Subtractor?

A Full Subtractor is a combinational logic circuit that performs binary subtraction by considering two input bits and a borrow input from the previous stage. It generates both the Difference and Borrow outputs, making it suitable for multi-bit subtraction.

## Behavioral Modeling

The Full Subtractor is implemented by describing its functional behavior using procedural statements. The Difference output is generated using XOR operations, while the Borrow output is determined through Boolean expressions.
