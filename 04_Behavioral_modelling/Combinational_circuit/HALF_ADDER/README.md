# Half Adder (Behavioral Modeling)

## Overview

This folder contains the behavioral implementation of a Half Adder using Verilog HDL. The Half Adder performs the addition of two single-bit binary numbers and produces a Sum and a Carry output.

## What is a Half Adder?

A Half Adder is a combinational logic circuit that adds two one-bit binary inputs. Since it does not accept a carry input from a previous stage, it is mainly used as a fundamental building block for designing larger arithmetic circuits such as Full Adders.

## Behavioral Modeling

The Half Adder is implemented by describing its functional behavior using procedural statements. The Sum is generated using the XOR operation, while the Carry is generated using the AND operation.
