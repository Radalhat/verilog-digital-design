# Comparator (Behavioral Modeling)

## Overview

This folder contains the behavioral implementation of 2-bit and 4-bit comparators using Verilog HDL. The designs compare two binary numbers and indicate whether one value is greater than, less than, or equal to the other.

## What is a Comparator?

A comparator is a combinational logic circuit that compares two binary inputs and produces outputs indicating their relationship. It is widely used in arithmetic units, processors, sorting circuits, and decision-making logic within digital systems.

## Behavioral Modeling

Behavioral modeling describes the functionality of the comparator without specifying the underlying hardware. The comparison is implemented using conditional statements to determine whether the first input is greater than, less than, or equal to the second input.

## Implementations Included

- 2-Bit Comparator
- 4-Bit Comparator

Each implementation includes an individual testbench for functional verification.

## Applications

Comparators are commonly used in:

- Arithmetic Logic Units (ALUs)
- Processor control units
- Digital sorting circuits
- Address comparison
- Data verification
- FPGA and ASIC digital systems

## Conclusion

This project demonstrates the behavioral implementation of binary comparators in Verilog HDL. These designs provide a practical understanding of digital comparison techniques and serve as essential components in arithmetic and control circuits.
