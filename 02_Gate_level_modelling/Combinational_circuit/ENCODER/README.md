# Encoder (Gate-Level Modeling)

## Overview

This folder contains the gate-level implementation of 4×2 and 8×3 encoders using Verilog HDL gate primitives. The designs convert a single active input line into its corresponding binary output code through logic gate connections.

## What is an Encoder?

An encoder is a combinational logic circuit that converts one active input out of many input lines into a binary code at the output. It performs the reverse operation of a decoder and is commonly used to reduce the number of data lines in digital systems.

## Gate-Level Modeling

Gate-level modeling represents a digital circuit using Verilog gate primitives such as `or`, `and`, `not`, `nand`, and `nor`. In these implementations, the encoder outputs are generated by directly connecting logic gates to produce the required binary code.

## Implementations Included

- 4×2 Encoder
- 8×3 Encoder

Each implementation includes an individual testbench for functional verification.

## Applications

Encoders are commonly used in:

- Keyboard encoding
- Interrupt handling
- Data compression
- Digital communication systems
- FPGA and ASIC digital designs
- Control and automation systems

## Conclusion

This project demonstrates the gate-level implementation of encoder circuits using Verilog HDL gate primitives. These designs illustrate how logic gates can be combined to generate binary codes from active input signals and provide a foundation for more advanced digital system design.
