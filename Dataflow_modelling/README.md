# Dataflow Modeling

## Overview

This folder contains digital circuits implemented using the dataflow modeling style in Verilog.

The designs use continuous assignment (`assign`) statements to describe how the outputs change based on the inputs. Each project includes the Verilog design, testbench, waveform, and a short README.

## Folder Structure

```text
Dataflow_Modeling/
│
├── Logic_Gates/
├── Combinational_Circuits/

```

## Summary

Working on these projects helped me become more comfortable using `assign` statements to implement digital circuits. It also improved my understanding of how dataflow modeling differs from gate-level, structural, and behavioral modeling.
